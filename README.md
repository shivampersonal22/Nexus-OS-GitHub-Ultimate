Nexus-OS GitHub Ultimate is a comprehensive Firefox add-on that combines the best features of three popular GitHub enhancement tools into a single, lightweight package.<img width="1366" height="643" alt="image" src="https://github.com/user-attachments/assets/d4db8cbd-5e1c-4c62-8949-03c78fec8f36" />


**Key Features:**

🌳 **File Tree Navigation (Octotree DNA)**
- Recursive repository file tree with instant navigation
- GitHub GraphQL API integration (supports private repos)
- Lazy-loading with IntersectionObserver for performance
- Collapsible and resizable sidebar
- File size display for each entry
- One-click file navigation without page reloads

⚡ **UI Enhancements (Refined GitHub DNA)**
- Quick-action bar on Issue/PR pages
- Sticky conversation sidebars for better context
- Keyboard shortcuts (G+S to star repository)
- Scroll-triggered UI elements with smooth animations
- Professional Nexus-OS dark theme

📥 **Power Utilities (Enhanced GitHub DNA)**
- One-click file downloads directly from listings
- Automatic file size calculation and display
- Code copy buttons on all code blocks
- Repository and file metadata display
- Dynamic content detection and updates

**Technical Highlights:**
- Manifest V2 (Firefox standard)
- Zero external dependencies
- ~15KB total size (optimized)
- <500ms load time
- 15-25MB memory usage
- WCAG AA accessibility compliant
- No data collection or tracking

**Perfect for:**
- Developers who spend hours on GitHub
- Teams managing multiple repositories
- Anyone wanting a more powerful GitHub experience
- Users who love efficient workflows

**Privacy:**
This add-on collects no data, includes no tracking, and respects your privacy completely. All processing happens locally on your computer.

**Permissions Explained:**
- Storage: Remembers sidebar state
- GitHub access: Fetches repository data
- Active tab detection: Only works on GitHub.com
- Web requests: For API communication
